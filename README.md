## Docker
Run `docker compose up` to start the project using Docker.

Docker starts:
- The PostgresSQL database on port 5432.
- The Java/Spring Boot backend on port 8080.
- The Angular frontend on port 4200.

Credentials for the database:
- Username: `postgres`
- Password: `1234`
- Database name: `bank`
