## Build
Run `./gradlew build` to build the project.

## Development server
Run `./gradlew bootRun` for a dev server.

Http server [http://127.0.0.1:8080/bank/v1](http://127.0.0.1:8080/bank/v1).

## Tests
Run `./gradlew clean test` to execute the tests.
